## 💬 What It Does
ByteBot is a terminal-based chatbot that understands common questions
and replies with helpful, friendly responses. Built with pure Python — no APIs needed!

---

## ✨ Features
- Responds to greetings, questions, and jokes
- Clean pattern-matching AI logic
- Easy to extend with new responses
- Beginner-friendly code with clear structure
